# Personal Portfolio Website

My personal portfolio website showcasing my projects, skills, blog posts, and professional experience. Built with Next.js 14, TypeScript, and Tailwind CSS.

## Live Site

**[resume-website-theta-blue.vercel.app](https://resume-website-theta-blue.vercel.app/)**

---

## Features

- **Responsive Design** — Fully mobile-responsive layout with dark mode support.
- **Project Showcase** — Detailed project pages with problem/solution format and live demos.
- **AI Chat Widget** — Integrated RAG-powered chatbot that answers questions about my CV.
- **Blog Section** — Technical blog posts about AI, ML, and web development.
- **Contact Form** — Direct contact form with email integration.
- **SEO Optimised** — Full meta tags, Open Graph, and Twitter Card support.
- **Resume Download** — Downloadable CV in DOCX format.

## Getting Started

### Prerequisites

- Node.js 18+
- pnpm (recommended) or npm

### Installation

```bash
# Clone the repository
git clone https://github.com/moqayyubok/resume_website.git
cd resume_website

# Install dependencies
pnpm install

# Start development server
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

### Production Build

```bash
pnpm build
pnpm start
```

### Docker

```bash
docker-compose up --build
```

---

## Deployment

This site is automatically deployed to **Vercel** on every push to the `main` branch.
